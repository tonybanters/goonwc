* GoonWC — Goon Wayland Compositor

Minimal, but functional wayland compositor written in C, and configured in Goon. No Wlroots, just a simple open wayland library implementation.

Uses OWL (Open Wayland Library) instead of wlroots. OWL and GoonWC are being concurrently developed. OWL's vision can be more understood by reading OWL's source code and readme.

* Installation
** NixOS (with Flakes)
DWC includes a =flake.nix= for development. Clone the repo and build with:

#+begin_src sh
git clone https://github.com/tonybanters/goonwc
cd dwc
nix develop -c make
#+end_src
